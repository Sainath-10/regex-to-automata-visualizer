What is Thompson's Construction?
Thompson's Construction Algorithm

Thompson's Construction is a systematic algorithm that converts any regular expression into an equivalent NFA. It works by breaking the regex into its atomic pieces and combining them using a fixed set of NFA fragment templates вҖ” one per regex operator.

The resulting NFA has exactly one start state and one accept state, with Оө (epsilon) transitions used to connect fragments together.
