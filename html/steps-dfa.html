What is Subset Construction?
Subset Construction Algorithm

Subset Construction (also called Powerset Construction) converts an NFA to an equivalent DFA. The key idea is that each DFA state represents a set of NFA states — specifically, all NFA states the machine could simultaneously be in after reading the same input.

Since a DFA must be deterministic, we compute — for every possible set of NFA states and every input symbol — exactly where we end up. This eliminates non-determinism.

ε-Closure — The Foundation

The ε-closure of a set of states S is the set of all states reachable from any state in S using only ε (epsilon) transitions — without consuming any input symbol.

ℹ

ε-closure(S) = S ∪ { all states reachable from S via one or more ε-transitions }

We also define move(S, a) as the set of states reachable from any state in S via exactly one transition on input symbol a.

The full transition in the DFA is then: δ_DFA(S, a) = ε-closure( move(S, a) )
